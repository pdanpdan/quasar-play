# Quasar Project

This is a project template using [Vite](https://vitejs.dev/). It requires [Node.js](https://nodejs.org) v18+.

To start in DEV mode:

```sh
pnpm install
pnpm dev

# if using yarn:
yarn
yarn dev
```

To build:

```sh
pnpm install
pnpm build

# if using yarn:
yarn
yarn build
```


To build and preview result:

```sh
pnpm install
pnpm build
pnpm preview

# if using yarn:
yarn
yarn build
yarn preview
```
